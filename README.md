# Battleship Game

Classic Battleship game built with JavaScript and Webpack.

## Live Demo

[https://battleship-game-nine-dun.vercel.app](https://battleship-game-nine-dun.vercel.app)

## Tech Stack

- JavaScript (ES6+)
- Webpack
- HTML5/CSS3
- Jest (testing)

## Local Development

```bash
npm install
npm run dev
Run Tests
bash
npm test
Build for Production
bash
npm run build
Deploy to Vercel
bash
cd dist
vercel --prod
When prompted:

Set up and deploy? Y

Link to existing project? N (for new project)

Project name? battleship-game

Directory? ./

Deployment Notes
This project uses Webpack. The dist folder contains the static files for deployment. Deploy from the dist directory, not the root.

Features
Drag-and-drop ship placement

Turn-based gameplay

Computer AI opponent

62 TDD tests

Responsive design
